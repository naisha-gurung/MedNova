# 🏥 MedNova

A lightweight **Hospital Management System** built with **Node.js** and **Express.js**.  
This web application provides a simple interface for managing **patients, doctors, and appointments** with full **CRUD operations**.

Data is stored in **JSON files**, making it easy to set up and run **without a database**.

---

## ✨ Features

- **Patient Management**  
  Register and manage patient records.

- **Doctor Management**  
  Add and track doctors along with their specializations.

- **Appointment Scheduling**  
  Book and manage appointments between patients and doctors.

- **Dashboard**  
  Provides system overview with statistics and status.

- **REST API**  
  Backend API for performing data operations.

- **Responsive Design**  
  Works smoothly on both **desktop and mobile devices**.
